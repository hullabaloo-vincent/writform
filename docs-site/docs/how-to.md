# How-to

## Run a writing session

1. Open **Sessions**, pick the channel, **+ New session**, name it.
2. **+ Add a prompt** — write it with the formatting toolbar, optionally set
   a timer in minutes, **Add prompt**.
3. Press **Start** when everyone's ready. Members write in their own private
   editor; the side chat stays open the whole time.
4. **Stop** early if you like, or let the timer end the prompt — everyone's
   writing is revealed. Add more prompts or **End session**.

## Invite people to a group

Group name header → **Invite** (admins only) → share the temporary code.
They redeem it from the **+** button in the chat sidebar.

## Add a custom emote

Composer → smiley button → **+ Add emote** (admins only) → upload an image
and name it. Everyone can then type `:name:`.

## Join voice

Click a channel under **VOICE** in the chat sidebar; grant microphone
access. Mute and leave from the pill in the status bar — it stays with you
in every app.

## Share a note

Notes → open the note → **Share** → pick a friend. They get a card in your
DM with a **Save to vault** button.

## Use the canvas

- Click tools in the bottom bar to place stickies / text / frames.
- **Paste** an image or a URL directly onto the board.
- Shift-click (or shift-drag on empty space) to select several elements.
- Select a connector to change its anchors, dash style, and end decorations.

## Force-logout a device

Settings → **Devices** shows every login for your account — revoke any of
them. Server admins can force-logout any user from the **Admin** tab.
